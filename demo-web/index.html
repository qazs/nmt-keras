Interactive neural machine translation

Demo of an interactive neural machine translation (INMT) system built with the toolkit NMT-Keras.

Instructions:

  1. Upload or choose a file with sentences in English to translate.
  2. Once loaded, click on each "Translate" button for obtaining the automatic translation in Spanish.
  3. If the translation is not correct, you can make corrections:
    1. Use the mouse or the keyboard arrows to select the position where to introduce the correction.
    2. Type the correction.
    3. Type "/" for introducing an end-of-sentence symbol.
  4. The system will react to these interactions, providing alternative translation hypotheses.
  5. Once you are satisfied with the translation, click in the "Validate" button and go to the next sentence.
  6. If Learn from sample is activated, the system will use this sentence pair for improving the model.

More info

All systems were developed with the open-source toolkit NMT-Keras.
Interactive machine translation is a framework that aims at diminishing the effort required for correcting translations from a machine translation system. For more info, look the following papers:

NMT-Keras: a Very Flexible Toolkit with a Focus on Interactive NMT and Online Learning
Álvaro Peris, Francisco Casacuberta.
In The Prague Bulletin of Mathematical Linguistics vol. 111, pp. 113-124, 2018.

Online Learning for Effort Reduction in Interactive Neural Machine Translation
Álvaro Peris and Francisco Casacuberta.
In arXiv:1802.03594, 2018.
